💡 Insight Fundamental: Toda aplicação web moderna é um SPA (Single Page Application) que troca apenas JSON com APIs REST.

Por que isso importa para seu TCC?

  • 95% dos projetos acadêmicos usam essa arquitetura
  • Empresas exigem conhecimento de APIs REST + React/Vue
  • Deploy fácil em Vercel/Netlify (5 minutos)

⚙️ Protocolos e Tecnologias Essenciais

HTTP/HTTPS → REST APIs

// GET - Buscar dados GET /api/users // POST - Criar POST /api/users { "name": "João", "email": "joao@email.com" } // PUT - Atualizar PUT /api/users/123 { "name": "João Silva" } // DELETE - Remover DELETE /api/users/123

📦 JSON: O coração das APIs

{{ "user": {{ "id": 123, "name": "João Silva", "email": "joao@email.com", "posts": [ {{ "id": 1, "title": "Meu TCC" }}, {{ "id": 2, "title": "Web Moderna" }} ] }} }}
⚠️ Erro Comum: Não confunda POST /users (cria novo) com GET /users (lista todos).
